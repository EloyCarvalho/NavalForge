# Changelog

## R0 — Motor de Embarcações Semelhantes

Primeira release técnica do motor Python para benchmark preliminar por embarcações semelhantes.

Inclui:

- leitura de base NavalForge em `.xlsx` ou `.csv`;
- cálculo de similaridade;
- TOP N semelhantes;
- faixas recomendadas;
- regressão linear preliminar;
- relatório em Markdown e JSON;
- exemplo DGS/FRB 8,5 m.

Limitações:

- ferramenta preliminar;
- regressão deve ser usada apenas de forma segmentada;
- não substitui responsabilidade técnica, norma aplicável, cálculo completo, CFD, FEA, ensaio ou prova de mar.
