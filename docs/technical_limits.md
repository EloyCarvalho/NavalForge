# Limitações técnicas

Esta versão é um MVP técnico preliminar.

Não substitui projeto naval completo, responsabilidade técnica, norma aplicável, modelo 3D validado, curvas hidrostáticas reais, curva GZ, análise de pesos detalhada, ensaio de tanque, CFD, FEA ou prova de mar.

O módulo `savitsky_screening` ainda é um scaffold de triagem e não deve ser tratado como implementação Savitsky final.
