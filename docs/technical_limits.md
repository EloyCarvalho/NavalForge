# Limitações técnicas

Esta versão é um MVP técnico preliminar.

## O que ela não substitui

- projeto naval completo;
- responsabilidade técnica profissional;
- norma aplicável;
- modelo 3D validado;
- curvas hidrostáticas reais;
- curva GZ;
- análise de pesos detalhada;
- ensaio de tanque, CFD, FEA ou prova de mar.

## Ponto crítico

O módulo `savitsky_screening` ainda é um scaffold de triagem. Ele usa variáveis típicas de embarcações planantes, mas não deve ser tratado como implementação Savitsky final sem revisão bibliográfica, validação numérica e comparação com casos conhecidos.
