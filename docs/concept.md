# Conceito do NavalForge

O NavalForge é uma plataforma experimental para apoiar a fase inicial de projeto naval.

A ideia é transformar cálculos isolados em um fluxo estruturado de avaliação de alternativas, com foco em:

- exploração do espaço de projeto;
- estimativas preliminares;
- suporte à decisão;
- ensino de arquitetura naval;
- preparação para análises mais rigorosas: Orca3D, CFD, FEA, prova de mar e normas aplicáveis.

O sistema deve apoiar o julgamento de engenharia, não substituir o engenheiro.
