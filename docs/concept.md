# Conceito do NavalForge

O NavalForge é uma plataforma experimental para apoiar a fase inicial de projeto naval.

A proposta é transformar cálculos isolados em um fluxo estruturado de avaliação de alternativas, com foco em exploração do espaço de projeto, estimativas preliminares e suporte à decisão.

O sistema deve apoiar o julgamento de engenharia, não substituir o engenheiro.
