# O Arquiteto Naval

Site estático inicial do projeto **O Arquiteto Naval**.

Conteúdo publicado via GitHub Pages a partir da pasta `docs/`.

## Estrutura

- `index.html` — site principal em página única.
- `robots.txt` — instrução básica para mecanismos de busca.
- `sitemap.xml` — mapa do site.
